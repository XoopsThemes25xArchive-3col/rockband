README :: rockband  XOOPS THEME 

*********************************************


In the Zip file you will find the following elements:

1. PSD Logo Fonts: font file for the site name and Logo.

2. XOOPS Theme: the theme files, the folder rockband should be placed inside the themes folder of your xoops, then activate the theme from the admin side in system > preferences > general settings.


**includes the following templates: system_redirect.html, system_siteclosed.html 

if you use xoops version 2.0.14 or higher they will automatically be visible to you without any need of updating anything; if you use an older xoops version you will have to place the files inside the respective folders of the module's templates directory, then update the module from the module administration panel.
